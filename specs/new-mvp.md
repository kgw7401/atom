🥊 1. 제품의 본질 (가장 중요)

👉 이건 “AI 코치”가 아니라

혼자 복싱할 때 켜는 리듬 기반 훈련 앱

⸻

해결하려는 문제
• 혼자 운동하면 지루함
• 매번 같은 드릴 반복
• 뭘 해야 할지 고민됨

⸻

핵심 가치

👉 “생각 안 해도 운동이 계속 이어진다”

⸻

🎯 2. MVP에서 검증할 것 (핵심 가설)

❌ “AI 드릴이 좋은가?”
✅ “이걸 틀어놓고 운동하는 습관이 생기는가?”

⸻

성공 기준
• 3번 이상 자발적 재사용
• “혼자 할 때 이거 켜게 된다”라는 피드백

⸻

🧠 3. MVP 전략 (중요한 방향 전환)

❌ 처음 생각
• 매번 LLM으로 드릴 생성

⸻

✅ 바뀐 방향

👉 미리 만든 세션 + 조합으로 다양성 확보

⸻

⚙️ 4. 콘텐츠 구조 (핵심 설계)

세션 구성

세션 = 여러 블록 조합

[워밍업]
[콤비네이션]
[디펜스]
[압박]
[마무리]

⸻

실행 시
• 블록 랜덤 조합
• 콤비 변형
• 템포 변화

👉 매번 다른 느낌

⸻

🔁 5. 다양성 만드는 방법

① 콤비 변형
• 원투 → 원원투 / 원투훅 / 원투백

⸻

② 템포 변화
• fast / slow / 끊어치기

⸻

③ 코치 멘트
• “좋아!”
• “계속!”
• “가드 올려!”

👉 체감 다양성 크게 증가

⸻

💰 6. 비용 전략

❌ 로그인으로 막기
• 전환율 떨어짐
• 근본 해결 아님

⸻

✅ 해결 방법
• 세션 미리 생성
• 재사용
• (옵션) 하루 1회 생성

⸻

👉 MVP에서는 LLM 거의 안 써도 됨
