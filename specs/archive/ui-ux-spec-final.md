음성 코칭을 프로덕트의 심장(Core)으로 삼고, 글러브를 낀 유저의 물리적 제약까지 완벽하게 고려한 **[오디오 중심 와이어프레임]**을 설계해 드립니다.

이 설계의 원칙은 **"말은 길게, 화면은 크게, 터치는 단순하게"**입니다.

---

## 1. 진입 및 온보딩 (Entry & Onboarding)

_비전: "코치님이 당신을 파악하는 첫 만남"_

### [Screen 1: 로그인 & 시작]

- **비주얼:** 샌드백을 치는 역동적인 실루엣.
- **버튼:** `[ 구글로 3초 만에 시작하기 ]`
- **UX:** 가장 빠른 진입을 유도하여 훈련 의지가 꺾이기 전에 온보딩으로 넘깁니다.

### [Screen 2: 프로필 설정 (온보딩)]

- **질문 1:** "당신의 복싱 수준은?" (입문/중급/선수)
- **질문 2:** "어떤 세션 위주로 수행하고 싶으세요" (스피드/체력/...)
- **결과:** AI 코치가 목소리로 첫 인사를 건넵니다. _"반가워요! 당신의 복싱 데이터를 관리할 파트너 AI 코치입니다."_

|                                       |

## 3. 세션 프리뷰 & 의도 입력 (The Prep)

_비전: "글러브 끼기 전, 코치와의 짧은 브리핑"_

### [Screen 3: 훈련 의도 전달]

- **AI 코치:** _"오늘 훈련에서 특별히 신경 쓰고 싶은 게 있나요? 지난번엔 어깨가 무겁다고 하셨죠."_
- **UI:** \* "오늘은 카운터 연습할래"라고 적음
- 추천 키워드 태그: `#카운터`, `#체력지옥`, `#위빙중점`

- **확정:** AI가 루틴을 짠 후 오디오로 요약해 줍니다. _"알겠습니다. 카운터 위주 6라운드 구성할게요. 글러브 끼고 시작 버튼을 눌러주세요."_

---

## 4. 훈련 진행 (The Active Workout)

_비전: "화면은 거들 뿐, 핵심은 오디오"_

### [Screen 4: 훈련 중 화면 (오디오 모드)]

```text
+---------------------------------------+
| R 3 / 6                [ 🔊 볼륨 최대 ] |
+---------------------------------------+
|                                       |
|               01 : 45                 |  <-- 화면의 40% (잔여 시간)
|                                       |
+---------------------------------------+
|                                       |
|           JAB - JAB - CROSS           |  <-- 현재 지시어 (대문자)
|                                       |
+---------------------------------------+
|                                       |
|     [ ⏸️ 일시정지 ]      [ ⏭️ 다음 ]      |  <-- 50:50 분할 거대 버튼
|                                       |
+---------------------------------------+

```

- **오디오 인터페이스:**
- **명령:** "잽! 잽! 크로스!" (리듬감 있는 TTS)
- **격려:** "좋아요! 지금 페이스 끝까지 유지하세요!"
- **스퍼트:** 종료 30초 전, 빠른 비트의 배경음과 함께 "자, 연타! 더 빠르게!"

- **컬러 UX:** \* 운동 중: **그린 배경**
- 30초 전: **레드 점멸**
- 휴식 중: **블루 배경**

## 5. 종료 및 보상 (Summary & Reward)

_비전: "데이터로 증명된 나의 승리"_

### [Screen 5: 성장 요약 및 카드]

- **AI 오디오 총평:** _"수고하셨어요! 오늘 요청하신 카운터 연습 덕분에 펀치 정확도가 15% 상승했습니다."_
- **성장 카드 비주얼:**
- 오늘의 펀치 수: `1,230`
- 최고 속도 배지: `TOP SPEED ⚡`

- **액션:** `[ 📸 성장 카드 공유하기 ]` (인스타그램용 카드 생성)
